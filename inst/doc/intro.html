Overview

StatComp21082 is a simple R package developed to find the shortest path between two vertex and use Gauss function as kernel density estimation for random sample of a two-component normal mixture. Two functions are considered, namely, floyd (find the shortest path between two vertex) and kerden (use Gauss function as kernel density estimation for random sample).

Finding the shortest path using floyd

In order to find the shortest path between two vertex,let matrix D be the shortest distance between i to j and matrix path be the successor to the vertex i on the shortest path between i to j .

The source R code for floyd is as follows:

floyd=function(A){
  n<-nrow(A)
  D<-A
  path<-matrix(0,n,n)
  for(i in 1:n){
    for(j in 1:n){
      if(is.finite(D[i,j])==T){path[i,j]=j}
    }
  }
  for(k in 1:n){
    for(i in 1:n){
      for(j in 1:n){
        if(D[i,k]+D[k,j]<D[i,j]){
          D[i,j]=D[i,k]+D[k,j];
          path[i,j]=path[i,k]
        }
      }
    }
  }
  return(list(path=path,D=D))
}

Using Gauss function as kernel density estimation using kerden

In order to estimate kernel density for random sample of a two-component normal mixture,random sample of a two-component normal mixture is generated by dep,and Use Gauss function as kernel density estimation using kerden.

The source R code for kerden is as follows:

kerden=function(x,h){
  x=sort(x)
  n=length(x);s=0;t=0;y=0
  for(i in 2:n)
    s[i]=0
  for(i in 1:n){
    for(j in 1:n)
      s[i]=s[i]+exp(-((x[i]-x[j])^2)/(2*h*h))
    t[i]=s[i]
  }
  for(i in 1:n)
    y[i]=t[i]/(n*h*sqrt(2*pi))
  z=complex(real=x,imaginary=y)
  return(z)
}

The source R code for dep is as follows:

dep=function(mu,sigma,p){
  n=30 
  i=sample(1:2,size=n,replace=TRUE,prob=c(p,1-p))
  x=rnorm(n, mu[i], sigma[i])
  m=5000
  xt=numeric(m)
  u=runif(m)
  a=1 
  b=1
  y<-rbeta(m,a,b) 
  xt[1]=0.5
  for (i in 2:m) {
    fy<-y[i]*dnorm(x,mu[1],sigma[1]) +
      (1-y[i])*dnorm(x,mu[2],sigma[2])
    fx<-xt[i-1]*dnorm(x,mu[1],sigma[1])+
      (1-xt[i-1])*dnorm(x,mu[2],sigma[2])
    r<-prod(fy/fx)*
      (xt[i-1]^(a-1)*(1-xt[i-1])^(b-1))/
      (y[i]^(a-1)*(1-y[i])^(b-1))
    if(u[i]<=r) xt[i]<-y[i] 
    else xt[i]<-xt[i-1]
  }
  return(xt)
}
